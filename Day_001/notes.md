## 📚 Topics Learned

### 1. Variables
- Variables are used to store data.
- Example:
```python
name = "Rohan"
age = 22
```

### 2. Data Types
- int
- float
- str
- bool

Example:
```python
x = 10
price = 99.99
name = "Rohan"
is_student = True
```

### 3. Input & Output

Input:
```python
name = input("Enter your name: ")
```

Output:
```python
print("Hello", name)
```

### 4. Operators

Arithmetic Operators:
- +
- -
- *
- /
- %
- **
- //

Comparison Operators:
- ==
- !=
- >
- <
- >=
- <=

### 5. if-else

Example:
```python
age = 18

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

### 6. Functions

Example:
```python
def add(a, b):
    return a + b
```

Benefits:
- Reusable code
- Better readability
- Easy maintenance
