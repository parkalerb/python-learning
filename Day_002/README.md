## 📖 What is a Function?

A function is a reusable block of code that performs a specific task.

Example:

```python
def add(a, b):
    return a + b
```

Functions help reduce code duplication and improve readability.

---

## 🔄 Return vs Print

### print()

- Displays output on the screen.
- Cannot be reused in calculations.

Example:

```python
print(10 + 20)
```

---

### return

- Sends a value back to the caller.
- Can be stored and reused.

Example:

```python
result = add(10, 20)
```

## 🎤 Interview Questions

### What is a Function?

A function is a reusable block of code used to perform a specific task.

---

### Function vs Method

Function:
- Independent block of code.

Method:
- Function associated with an object or class.

---

### Local vs Global Variables

Local Variable:
- Declared inside a function.
- Accessible only within that function.

Global Variable:
- Declared outside functions.
- Accessible throughout the program.

---

### Return vs Print

print():
- Displays output only.

return:
- Sends a value back for further use.

---

### Why use Functions?

- Code Reusability
- Better Readability
- Easy Debugging
- Modular Programming
- Easy Maintenance
